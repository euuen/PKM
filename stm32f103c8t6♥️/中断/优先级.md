# 优先级

优先级有抢占优先级和响应优先级。

stm32只用了4bits来存储这两个优先级。一般是不设置响应优先级，4bits全部用来设置抢占优先级。为什么？因为我感觉响应优先级比较鸡肋。具体原理可以参考下面的`参考资料1`。

## 设置优先级

```c
HAL_NVIC_SetPriority(EXTI1_IRQn, 1, 0);
```

第一个参数书IRQn，第二个参数是抢占优先级，第三个参数是响应优先级。

数字越小，优先级越高。
