# 云嵌第四次电控考核

## 时间

12月2日-12月14日晚上9点

## 考核地点

卓工中厅（暂定）

## 学习任务

1. 根据江协科技视频学习ADC，DAC的内容，掌握 ADC，DAC 的基本原理和标准库使用
1. 学习 DMA 的使用，掌握 DMA 的标准库配置，了解 DMA 中断的内容
1. 学习了解 IIC 的通信流程，掌握标准库的软件 IIC 和硬件 IIC 的配置
1. 学习 SPI 通信的内容，掌握标准库的 SPI 通信的内容

## 考核任务

- 任务一
  - 使用 ADC 加 DMA 来完成红外测距，OLED 显示距离
- 任务二
  - 使用 DMA 加串口完成电脑通过串口发送不定长的数据，通过 dma 转送到内存中，然后再用 OLED 屏幕打印出来
- 任务三
  - 使用 SPI 配置OLED模块，在OLED屏幕第一行显示十六进制数0x01，第二行显示十进制数100，第三行显示字符串“helloworld”（可以尝试显示自己的名字）。需理解OLED驱动，或自己写OLED驱动，到时会提问
- 任务四
  - 使用软件模拟 IIC 通信配置MPU6050模块，通过加速度原始值解算出滚转角和俯仰角（解算处理不做要求）。当 滚转角绝对值大于30度时点亮红灯 ，当俯仰角绝对值大于30度时点亮绿灯，LED灯的颜色可任意选择，不局限于红色和绿色，

  #### 注：任务三需使用 **SPI 完成**，请提前购买**支持 SPI 通信的 OLED 模块**

  

## 提示

任务二的串口接收的数据，可以自行添加帧头和帧尾，也可以选择不使用固定的帧头和帧尾完成
