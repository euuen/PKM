# 6.X版本挂载物理硬盘

# <span style="color: blue;">挂载物理硬盘</span>

```sh
VBoxManage.exe internalcommands createrawvmdk -filename "D:\archlinux.vmdk" -rawdisk \\.\PhysicalDrive2
```

# 7.X版本挂载物理硬盘

```sh
VBoxManage createmedium disk --filename "C:\VMs\Kylin\hd2.vmdk"  --format=VMDK --variant RawDisk --property RawDrive=\\.\PhysicalDrive2
```

linux上
```sh
sudo VBoxManage createmedium disk --filename "～/vbox.vmdk" --format=VMDK --variant RawDisk --property RawDrive=/dev/sda
```
